В Python циклы for и while используются для выполнения повторяющихся операций. Вот примеры их использования и основные различия:

1. **Цикл for:**
   - Синтаксис:

     for элемент in последовательность:
         # блок кода

   - Пример использования:

     fruits = ["яблоко", "банан", "апельсин"]
     for fruit in fruits:
         print(fruit)

   - В цикле for переменная fruit последовательно принимает значения из списка fruits.

2. **Цикл while:**
   - Синтаксис:

     while условие:
         # блок кода

   - Пример использования:

     count = 0
     while count < 5:
         print(count)
         count += 1

   - Цикл while выполняется, пока условие истинно.

**Различия между for и while в Python:**
- for используется, когда заранее известно количество повторений (например, для итерации по списку), а while используется, когда количество повторений зависит от условия.
- В цикле for переменная перебирает элементы последовательности (список, кортеж, строку), а в цикле while проверяется условие.
- Цикл for более предпочтителен, когда известно количество итераций, в то время как цикл while удобен, когда необходимо повторять операции до выполнения определенного условия.

Оба типа циклов могут быть полезны в различных ситуациях, и выбор между ними зависит от конкретной задачи, которую вы хотите решить.
