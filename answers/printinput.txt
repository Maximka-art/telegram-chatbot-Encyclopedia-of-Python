В Python команда print используется для вывода информации на экран, а функция input используется для ввода данных пользователем с клавиатуры. Вот более подробно:

1. **Print:**
   - Синтаксис: print(выражение)
   - Пример использования:

     print("Привет, мир!")
     x = 10
     print("Значение переменной x равно:", x)


2. **Input:**
   - Синтаксис: input(prompt)
   - Пример использования:

     name = input("Введите ваше имя: ")
     print("Привет,", name)

     age = int(input("Сколько вам лет? "))
     print("Вам", age, "лет.")


   Функция input позволяет пользователю ввести данные с клавиатуры, а возвращаемое значение будет строкой. Если нужно преобразовать введенное значение в другой тип данных (например, целое число), можно использовать соответствующие функции преобразования данных, например int() или float().

Таким образом, комбинация print и input позволяет создавать интерактивные программы, которые выводят информацию на экран и запрашивают данные у пользователя для дальнейшей обработки.
