В Python обработка исключений осуществляется с помощью конструкции try-except. Пример:
try:
    # код, который может вызвать исключение
    result = 10 / 0
except ZeroDivisionError:
    # обработка исключения
    print("Деление на ноль!")

В данном примере, если при выполнении кода result = 10 / 0 произойдет деление на ноль, то будет вызвано исключение ZeroDivisionError, которое будет обработано блоком except, выводящим сообщение "Деление на ноль!".
Также можно использовать блоки try-except для обработки разных типов исключений:
try:
    # код, который может вызвать исключение
    result = int(input("Введите число: "))
except ValueError:
    # обработка исключения ValueError
    print("Ошибка! Необходимо ввести число.")
except ZeroDivisionError:
    # обработка исключения ZeroDivisionError
    print("Деление на ноль!")

Кроме того, с помощью блока try-finally можно выполнить некоторый код после обработки исключения:
try:
    # код, который может вызвать исключение
    result = 10 / 0
except ZeroDivisionError:
    # обработка исключения
    print("Деление на ноль!")
finally:
    # выполнение кода в любом случае
    print("Конец программы")

Таким образом, блоки try-except позволяют обрабатывать исключения и не давать программе аварийно завершиться при возникновении ошибок.
