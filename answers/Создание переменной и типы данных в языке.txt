Для создания переменной в Python необходимо присвоить ей значение, используя оператор присваивания "=".
Пример:
x = 5
Python поддерживает следующие типы данных:
1. Числовые типы:
   - int (целые числа)
   - float (вещественные числа)
   - complex (комплексные числа)
2. Строковый тип:
   - str (строки)
3. Логический тип:
   - bool (логические значения True или False)
4. Списки (list)
5. Кортежи (tuple)
6. Множества (set)
7. Словари (dict)
8. None (отсутствие значения)

Примеры создания переменных разных типов данных:
# числовой тип
x = 5
y = 3.14
z = 2 + 3j

# строковый тип
name = "Alice"

# логический тип
is_true = True

# список
my_list = [1, 2, 3]

# кортеж
my_tuple = (4, 5, 6)

# множество
my_set = {1, 2, 3}

# словарь
my_dict = {"name": "Bob", "age": 30}

# None
empty_variable = None
