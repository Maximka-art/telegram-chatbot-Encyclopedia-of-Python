Для установки Python на компьютер, вам нужно выполнить следующие шаги:

Перейдите на официальный сайт Python (https://www.python.org/downloads/) и скачайте последнюю версию Python для вашей операционной системы.

Запустите установочный файл Python и следуйте инструкциям установщика.

Убедитесь, что вы отметили опцию "Add Python to PATH" в процессе установки, чтобы Python был доступен из командной строки.

После завершения установки, вы можете открыть командную строку и проверить установку Python, введя команду "python --version" или "python -V". Если установка прошла успешно, вы увидите версию Python.

Теперь у вас установлен Python на вашем компьютере и вы можете начать писать и запускать Python-скрипты.
Источник: https://docs.python.org/3/using/windows.html
